# 📸 Instagram Influencer Influence Analysis

## 📌 Project Overview
This project analyzes the top 200 Instagram influencers worldwide to understand the factors driving their influence. Beyond simple follower counts, we explore engagement rates, recent performance trends, and audience segments to identify "Rising Stars" and evaluate communication density.

## 📊 Key Analysis Focus
1. **Communication Density:** Evaluating influencers based on engagement rates relative to follower size.
2. **Growth Potential:** Identifying influencers whose recent performance (likes on new posts) exceeds their historical average.
3. **Composite Score:** Creating a balanced metric combining scale (followers) and momentum (recent engagement).

## 🛠️ Tech Stack
- **Languages:** Python (Pandas, Matplotlib, Seaborn)
- **Tools:** Jupyter Notebook, Tableau
- **Data Source:** Global Top Instagram Influencers Dataset

## 🚀 How to Run
1. Clone this repository.
2. Navigate to `notebooks/` and run the Jupyter Notebook to see the preprocessing steps.
3. Access the interactive dashboard via the link in `dashboard/tableau_public_link.md`.
